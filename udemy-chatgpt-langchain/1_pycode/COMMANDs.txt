# Install env manager
> pip3 install pipenv [Didn't work probably due to homebrew]
    > Used -> brew install pipenv

# Setup dependencies list
> Create a file in your project directory called Pipfile and Update the file content.

# Install the dependencies from the Pipfile
> pipenv install

# Create and enter a new environment
> pipenv shell

# Run
> python main.py
> python main.py --task 'multiply two numbers' --language javascript 

# Exit
> exit

# Re-enter
> pipenv shell

# Install a package
> pipenv install pprintpp
> Or, add a package manually in Pipfile and run 'pipenv install'
